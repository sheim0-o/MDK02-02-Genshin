GENSHIN IMPACT PROJECT

IMPLEMENTED FUNCTIONALITY:
The program is an information site on the game Genshin Impact.
Pages of different sections of the site with information 
were implemented: about the game, the plot of the game, 
key characters, regions in which the game action takes place.
________________________________________________________________

PROJECT FEATURES:
The main thing in the project is that the information is 
conveyed accurately in a very juicy beautiful design based on the game.
